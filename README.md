Este proyecto consiste en un termómetro basado en diodo de silicio 1N4005/1N4007 utilizado como sensor de temperatura sobre Arduino (Nano) + Pantalla OLED de 128x64 pixeles. El termómetro se basa en el diodo.
característica de que al aumentar la temperatura se reduce su tensión directa (Vf) en 2,2mV/°C.
