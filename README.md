Este proyecto consiste en un termómetro basado en diodo de silicio 1N4005/1N4007 utilizado como sensor de temperatura sobre Arduino (Nano) + Pantalla OLED de 128x64 pixeles. El termómetro se basa en la característica de que al aumentar la temperatura del diodo, se reduce su tensión directa (Vf) en 2,2mV/°C. El termómetro inspirado principalmente en el Sensor de Temperatura con un diodo de Instructables y otros sitios de internet.
