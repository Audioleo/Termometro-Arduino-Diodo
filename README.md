# Termómetro basado en diodos de silicio

Construye este termómetro utilizado un simple diodo de silicio, como el 1N4007/1N4005 como sensor de temperatura sobre una Placa Arduino (Nano) + Pantalla OLED de 128x64 pixeles. El termómetro se basa en la característica que tienen los diodos de reducir su umbral de tensión directa (Vf) al aumentar la temperatura. Este proyecto está inspirado en el trasductor de sensor con diodo de Instructables y otros sitios de internet.
